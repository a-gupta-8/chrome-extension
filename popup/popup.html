Arrays

Arrays are one of the most common data structures in computer science. Arrays are stored contiguously in memory, which means blocks of memory are allocated for them and there can be no gaps between the memory addresses of its values.

The elements in an array can be identified by their index which represents thier position in the array. Usually, the first element has an index 0, the second element has an index 1 and so on.

If an array extends past the memory block it's allocated, then a new array, of larger size, is created and the the values are copied over to the next array.

Similarly, in some cases, if the array becomes too small then a smaller sized array is allocated and the values copied over

Time Complexity (amortised)
Insert at beginning O(n)
Insert at end O(1)
Remove from beginning O(n)
Remove from end O(1)
Access index O(1)
